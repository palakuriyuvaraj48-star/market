# FreshMart

FreshMart is a production-ready React supermarket storefront inspired by modern grocery platforms. It includes product discovery, cart, wishlist, checkout, orders, profile tools, and an admin dashboard.

## Tech Stack

- React, Vite, Tailwind CSS
- React Router DOM
- Redux Toolkit
- React Query and Axios mock services
- Framer Motion
- React Hook Form

## Run Locally

```bash
npm install
npm run dev
```

Open the URL shown by Vite, usually `http://localhost:5173`.

## Useful Scripts

```bash
npm run build
npm run preview
npm run lint
```

## Features

- Mobile-first responsive UI with light and dark modes
- Search with debounce, suggestions, category, price, rating, and stock filters
- Product catalog, category pages, product detail pages, recently viewed items
- Cart drawer, cart page, quantity updates, save for later, dynamic totals
- Wishlist with move-to-cart
- Checkout with address form, coupons, payment method, and order creation
- Order history, order details, reorder
- Profile, addresses, saved payments, notifications, order tracking
- Admin analytics, sales chart, products, categories, orders, users, coupons
